# How this site works

  1. The project expects you to have a json file with contacts.
  2. A sample json can be found in the github repository
  3. Go to the contacts section.
  4. Upload the json file. The application will automatically read the contents of the JSON.
  5. Select the contact to send message to.
  6. Click on message button to send the message.
  7. The message tag is editable, you can edit it here. By default it will have a template message.
  8. Click on the send message button.
  9. You will get a confirmation alert message then taken to contacts page.
  10. Repeat step 5 to 8.

# View logs

  1. Go to the logs section.
  2. A list of past logs are autopopulated.
  3. The red otps denote the otp is not yet verified.
  4. Click on the stats section to view some statistics of the otp messages.
